# 🔐 Password Generator (Python)

A simple Python project that generates strong and random passwords using letters, numbers, and special characters. This project helped me practice Python fundamentals, functions, loops, conditional statements, and working with built-in modules.

## 📌 Features

* Generate random passwords
* User-defined password length
* Includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Special characters (!@#$%^&*)
* Ensures the password length is at least 4 characters

## 🛠️ Technologies Used

* Python 3
* `random` module
* `string` module

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the program:

```bash
python password_generator.py
```

4. Enter the desired password length.
5. The program will generate a random password.

### Example

```
=== Password Generator ===

Enter password length: 12

Generated Password:
A#7kP!2x@mQ9
```
