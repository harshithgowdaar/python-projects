# 🧮 Simple Calculator

A simple command-line calculator built with Python that performs basic arithmetic operations.

---

## 📌 Features

* Addition
* Subtraction
* Multiplication
* Division
* Exponential
* Floor division
* Modulus
* Easy-to-use menu
* Handles user input

---

## 🛠 Technologies Used

* Python 3

---

## 📚 Concepts Practiced

* Functions
* Conditional Statements (`if`, `elif`, `else`)
* User Input
* Variables
* Arithmetic Operators

---

## ▶️ How to Run

```bash
python simple_calculator.py
```
