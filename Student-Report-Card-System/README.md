# 🎓 Student Report Card System

A simple **Object-Oriented Python** project that simulates a student report card management system. The application allows users to create student records, add subject marks, calculate averages, determine pass/fail status, assign grades, and manage students within a classroom.

This project was built to practice core **Python OOP concepts** through a real-world example.

---

## 🚀 Features

* 👨‍🎓 Create student records
* 📚 Add marks for multiple subjects
* 📊 Calculate each student's average marks
* ✅ Determine Pass/Fail status
* 🏆 Automatically assign grades
* 📝 Generate a formatted report card
* 🏫 Manage multiple students in a classroom
* 📈 Calculate the overall classroom average
